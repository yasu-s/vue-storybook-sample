# vue-storybook-sample

## Overview

- Vue.js + Storybook sample.

## System requirements

- Node.js - 12.x
- Yarn - 1.17.x

## Used library

- vue - 2.6.x
- @storybook/vue - 5.3.x

## Usage

### 1. Download Sample

```bash
git clone https://github.com/yasu-s/vue-storybook-sample.git
```

### 2.  Installing packages

```bash
cd vue-storybook-sample
yarn
```

### 3. Launch sample application

```bash
yarn storybook
```
